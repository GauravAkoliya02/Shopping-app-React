🛒 Shopping Cart React App

A simple e-commerce web application built using React.js.

## 🚀 Features
- Product listing
- Add to cart
- Remove from cart
- Checkout page

## 🛠️ Tech Stack
- React.js
- JavaScript
- CSS

## 📌 Description
This project simulates an online shopping experience where users can browse products, add them to cart, and proceed to checkout.
